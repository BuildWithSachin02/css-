Display:flex,Flex(Parent Property)Justify-Content

  • The justify-content property in CSS is a parent property used in Flexbox to align and distribute flex items along the main axis of the flex container. The main axis is determined by the flex-direction property.
  • The flex-wrap: wrap property is applied to the flex container and controls whether flex items should be forced into a single line or allowed to wrap onto multiple lines.
