Display:flex

Formally, the display property sets an element's inner and outer display types. The outer type sets an element's participation in flow layout; the inner type sets the layout of children. Some values of display are fully defined in their own individual specifications; for example the detail of what happens when display: flex is declared is defined in the CSS Flexible Box Model specification.

Display:flex,Flex-Wrap

The flex-wrap CSS property sets whether flex items are forced onto one line or can wrap onto multiple lines. If wrapping is allowed, it sets the direction that lines are stacked.

The flex-wrap: wrap-reverse

Wrap-reverse property is set on the flex container and controls how items wrap when they run out of space. Instead of wrapping to a new line below the current line (as with flex-wrap: wrap), this property makes items wrap to a new line above the current line.

Display:flex, flex-Direction

The flex-direction property is applied to the flex container and defines the main axis along which flex items are placed.

This determines whether the items are laid out horizontally or vertically.
